## 📌 Piano Web Application

A virtual piano that plays sounds on key press or click.

---

## 🛠️ Tech Stack

HTML5 • CSS3 • JavaScript

---

## ✨ Features

* Simulates a piano with clickable keys.
* Plays sound on interaction.
* Demonstrates audio integration in the browser
